# Torn Xanax Bazaar Watcher

Tampermonkey userscript that reads Weav3r TornW3B bazaar data for Xanax and notifies when a listing is at or below the configured price.

Defaults:

- Item: Xanax, Torn item ID `206`
- Target price: `$820,000`
- Refresh interval: `30` seconds

Install `torn-xanax-bazaar-watcher.js` in Tampermonkey, open Torn, and use the `XANAX` side tab to adjust settings or reset alert history.
